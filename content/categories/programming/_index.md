Some time ago, I was writing lots of software both professionally and recreationally. For some reason, I write less software now. Therefore, in the absence of a `portfolio` section, here are some of the "recreational" programs I've authored:

- [NML](https://github.com/nomemory/neat-matrix-library) - a neat matrix library;
- [MockNeat](https://github.com/nomemory/mockneat) - this was (and still is) a faker library that some people actually used;
- [ServerNeat](https://github.com/nomemory/serverneat) - a Kotlin Web Server for mocking and stubbing REST APIs;
- [NeatChess](https://github.com/nomemory/neat-chess) - a Java library to interact with chess engines;
- [MapNeat](https://github.com/nomemory/mapneat) - a Kotlin library for JSON transformation (from one format to another);
- [microblog-c](https://github.com/nomemory/microblog-c) - a small blogging engine that makes your blog an executable;
- [lc3-vm](https://github.com/nomemory/lc3-vm) - an LC3 virtual machine implementation in a few lines of C code;
- [integers-snake](https://github.com/nomemory/integers-snake) - a snake game in C with a twist (all the memory is compressed into 4 integers);
- [SinusoidalTetris](http://localhost:1313/2024/04/24/the-sinusoidal-tetris/) - a Tetris game with sinusoids.
